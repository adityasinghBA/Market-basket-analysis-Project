# Market Basket Analysis and Product Recommendation

## Business Problem
This project analyzes customer purchasing patterns to identify products frequently bought together and improve product recommendation systems.

## Dataset Source
Dataset provided by Masai School.

## Dataset Description
The dataset contains transaction-level retail purchase data including:
- Invoice Number
- Product Description
- Quantity
- Customer ID

## Tools & Libraries Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- mlxtend

## Steps Performed
1. Imported libraries
2. Loaded dataset
3. Data cleaning
4. Basket matrix creation
5. Frequent itemset generation using Apriori
6. Association rule mining
7. Visualization of lift values
8. Product recommendation insights

## Data Cleaning Summary
- Removed missing values
- Removed duplicate rows
- Removed cancelled transactions
- Removed invalid quantities

## Business Insights
- Frequently purchased product combinations were identified.
- Strong association rules were generated using lift and confidence.
- Product recommendation opportunities were discovered.

## Final Recommendations
- Create combo offers for related products.
- Improve cross-selling strategies.
- Recommend associated products during checkout.
- Use association rules for personalized marketing.
